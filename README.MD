# Programador de Citas Médicas Virtuales

Este proyecto es una aplicación web para la gestión de citas médicas virtuales. Permite a los usuarios (administradores y pacientes) interactuar con un sistema de citas, mostrando información sobre doctores, gestionando citas y actualizando datos de los doctores.

## Características

- **Administración de Doctores**: 
  - Agregar, editar y eliminar doctores.
  - Mostrar lista de doctores con sus descripciones y horarios disponibles.
- **Gestión de Citas**:
  - Programar, editar y eliminar citas.
  - Visualizar citas programadas.
- **Interfaz de Usuario**:
  - Roles diferenciados para administradores y pacientes.
  - Formularios para agregar y editar doctores y citas.

## Tecnologías Utilizadas

  - HTML
  - CSS
  - JavaScript
